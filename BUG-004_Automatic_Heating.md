# BUG-004: Automatic Heating Issue

## Bug ID
BUG-004

## Title
POCO X7 Pro Automatically Heats During Light Usage

## Device Information
- Device: POCO X7 Pro
- OS Version: HyperOS 3.0.5.0

## Severity
Medium

## Priority
Medium

## Description
The device heats up automatically during normal usage without running heavy applications, gaming, or charging. The issue was observed after updating to HyperOS 3.0.5.0.

## Preconditions
- Device updated to HyperOS 3.0.5.0
- Normal room temperature
- No heavy apps running

## Steps to Reproduce
1. Unlock the device.
2. Use normal applications such as WhatsApp, Chrome, Instagram, or YouTube.
3. Continue usage for 10–15 minutes.
4. Observe device temperature.

## Expected Result
The device should maintain a normal temperature during light usage.

## Actual Result
The device becomes noticeably warm even during light usage.

## Impact
- User discomfort
- Increased battery consumption
- Possible performance degradation
